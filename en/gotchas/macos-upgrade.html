Nix is broken after macOS upgrade

Upgrading macOS is known to break Nix. When this happens, just uninstall Nix and then install it again.

  • Uninstall Nix
    • Run /nix/nix-installer uninstall
    • If that path above does not exist, follow these instructions to manually unintall Nix.
  • Reboot your Mac
